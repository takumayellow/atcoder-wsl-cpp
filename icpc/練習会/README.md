# 練習会 — チーム練習・模擬予選

チームで集まって解く練習・模擬国内予選用。AOJ の過去問や JAG 模擬予選をここで回す。

## 進め方
1. 回ごとにフォルダを作る（例: `2026-06-25-mock/`, `aoj-domestic-2023/`）
2. その中に問題ごと `A/ B/ C/ ...` を作り、`../../template/main.cpp` か `main.py` をコピー
3. **複数データセット入力**に慣れる（末尾 `0` 終端のループ）。本番と同じ形式で練習する
4. 解いた問題で使った定石は `../../library/` に還元（チームノートを育てる）

## 過去問・練習データの入手先
- **公式アーカイブ（IISF）**: 年度別の問題・解説・データセット
- **AOJ**（会津大学オンラインジャッジ）: 国内予選の移植版を実際にジャッジできる
- **JAG（jag-icpc.org）**: 模擬国内予選の問題・入出力データ

## 練習で必ず確認すること
- 手元 PC での**実行時間の体感**（本番は実行時間が効く問題がある）
- **出力の完全一致**（行末・改行・大文字小文字）
- 3人での**役割分担と1台運用**の練習（誰が打つ・誰が考える・誰が紙で詰める）
- コンパイラ/Python バージョンを**本番ジャッジに合わせてあるか**
